Some Equivalences involving conditional statements:
p → q = ¬ q →¬ p
p → q = ¬ p ∨ q
p ∨ q = ¬ p → q
p ∧ q = ¬ (p →¬ q)
¬ (p → q) = p ∧¬ q
(p → q) ∧ (p → r) = p → (q ∧ r)
(p → q) ∨ (p → r) = p → (q ∨ r)
(p → r) ∧ (q → r) = (p ∨ q) → r
(p → r) ∨ (q → r) = (p ∧ q) → r

Some Equivalences involving biconditional statements:
p ↔ q = (p → q) ∧ (q → p)
p ↔ q = ¬ p ↔¬ q
p ↔ q = (p ∧ q) ∨ (¬ p ∧¬ q)
¬ (p ↔ q) = p ↔¬ q
